# TypeScript 学习与练习项目

这是一个完整的 TypeScript 学习项目，包含从基础到高级的知识点讲解和练习。

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 类型检查
npx tsc --noEmit
```

## 📚 知识点目录

### 第一阶段：基础入门

| 章节 | 内容 | 难度 |

| 02-functions | 函数类型、可选参数、重载、this类型、回调函数 | ⭐⭐ |

| 章节 | 内容 | 难度 |

| 04-classes | 类基础、访问修饰符、继承、多态、抽象类 | ⭐⭐⭐ |

### 第三阶段：进阶提升

| 章节 | 内容 | 难度 |
|------|------|------|

## 📖 学习建议

1. **按顺序学习**：每个章节都有 README.md 说明该章节的知识点
2. **动手练习**：每个文件末尾都有练习题，尝试自己实现
3. **观察类型**：将鼠标悬停在变量上查看 TypeScript 推断的类型
4. **修改代码**：故意写错代码，观察 TypeScript 的错误提示
