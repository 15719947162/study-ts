# 03 - 接口与类型别名 (Interfaces & Type Aliases)

接口和类型别名是 TypeScript 中定义复杂类型的两种主要方式。

## 知识点概览

1. **接口基础**: interface 定义与使用
2. **类型别名**: type 关键字
3. **接口 vs 类型别名**: 两者的区别与选择
4. **接口继承**: extends 与多继承
5. **交叉类型与联合类型**: & 和 | 运算符
6. **索引签名与映射类型**: 动态属性定义

## 文件说明

- `01-interface-basics.ts` - 接口基础
- `02-type-alias.ts` - 类型别名
- `03-interface-vs-type.ts` - 接口与类型别名对比
- `04-interface-extends.ts` - 接口继承
- `05-union-intersection.ts` - 联合类型与交叉类型
- `06-index-signatures.ts` - 索引签名

## 学习目标

- 熟练使用接口定义对象类型
- 掌握类型别名的使用场景
- 理解联合类型和交叉类型的区别
- 学会使用索引签名处理动态属性
