# 04 - 类与面向对象编程 (Classes & OOP)

TypeScript 提供了完整的类支持，包括访问修饰符、抽象类、接口实现等面向对象特性。

## 知识点概览

1. **类的基础**: 属性、方法、构造函数
2. **访问修饰符**: public, private, protected
3. **继承与多态**: extends, super, 方法重写
4. **抽象类与接口**: abstract, implements
5. **静态成员**: static 属性和方法
6. **getter/setter**: 存取器属性

## 文件说明

- `01-class-basics.ts` - 类的基础
- `02-access-modifiers.ts` - 访问修饰符
- `03-inheritance.ts` - 继承与多态
- `04-abstract-classes.ts` - 抽象类
- `05-static-members.ts` - 静态成员
- `06-getters-setters.ts` - 存取器

## 学习目标

- 掌握 TypeScript 类的完整语法
- 理解访问修饰符的作用
- 学会使用抽象类和接口进行面向对象设计
- 掌握常见的设计模式实现
