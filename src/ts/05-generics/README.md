# 05 - 泛型 (Generics)

泛型是 TypeScript 中最强大的特性之一，允许创建可重用的组件，这些组件可以支持多种类型。

## 知识点概览

1. **泛型基础**: 泛型函数、泛型接口、泛型类
2. **泛型约束**: extends 关键字限制类型参数
3. **泛型工具类型**: Partial, Required, Pick, Omit 等
4. **条件类型**: 基于条件的类型推断
5. **映射类型**: 基于现有类型创建新类型
6. **infer 关键字**: 类型推断

## 文件说明

- `01-generic-basics.ts` - 泛型基础
- `02-generic-constraints.ts` - 泛型约束
- `03-utility-types.ts` - 内置工具类型
- `04-conditional-types.ts` - 条件类型
- `05-mapped-types.ts` - 映射类型
- `06-infer-keyword.ts` - infer 类型推断

## 学习目标

- 理解泛型的作用和使用场景
- 掌握泛型约束的使用
- 熟练使用内置工具类型
- 能够创建自定义工具类型
