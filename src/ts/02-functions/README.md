# 02 - 函数与类型 (Functions & Types)

TypeScript 为函数提供了强大的类型系统支持，包括参数类型、返回值类型、函数重载等。

## 知识点概览

1. **函数类型注解**: 参数类型与返回值类型
2. **可选参数与默认参数**: ? 和 = 的使用
3. **剩余参数**: ...args 的类型定义
4. **函数重载**: 同一函数多种调用方式
5. **this 类型**: 函数中 this 的类型控制
6. **回调函数类型**: 高阶函数的类型定义

## 文件说明

- `01-function-basics.ts` - 函数基础类型
- `02-optional-default-params.ts` - 可选参数与默认值
- `03-rest-params.ts` - 剩余参数与展开运算符
- `04-function-overloads.ts` - 函数重载
- `05-this-type.ts` - this 类型
- `06-callback-types.ts` - 回调函数与高阶函数

## 学习目标

- 掌握函数参数和返回值的类型注解
- 理解函数重载的应用场景
- 学会定义复杂的函数类型
