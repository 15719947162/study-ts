# 01 - 基础类型 (Basic Types)

TypeScript 提供了丰富的基础类型系统，这是学习 TypeScript 的第一步。

## 知识点概览

1. **原始类型**: string, number, boolean, null, undefined, symbol, bigint
2. **数组类型**: Array<T> 或 T[]
3. **元组类型**: [T1, T2, ...]
4. **枚举类型**: enum
5. **any 与 unknown**: 任意类型与未知类型
6. **void, never**: 函数返回类型
7. **类型断言**: as 语法

## 文件说明

- `01-primitives.ts` - 原始类型讲解与练习
- `02-arrays-tuples.ts` - 数组与元组
- `03-enums.ts` - 枚举类型
- `04-any-unknown.ts` - any 与 unknown 的区别
- `05-void-never.ts` - void 与 never 类型
- `06-type-assertion.ts` - 类型断言

## 学习目标

- 理解 TypeScript 的类型系统基础
- 掌握各种基础类型的使用场景
- 学会使用类型断言进行类型转换
