# 06 - 高级类型 (Advanced Types)

高级类型是 TypeScript 中最强大的特性，包括条件类型、映射类型、模板字面量类型等。

## 知识点概览

1. **条件类型**: T extends U ? X : Y
2. **infer 关键字**: 类型推断
3. **映射类型**: 基于已有类型创建新类型
4. **模板字面量类型**: 字符串类型操作
5. **类型守卫**: 运行时类型检查
6. **类型断言函数**: asserts 关键字

## 文件说明

- `01-conditional-types.ts` - 条件类型
- `02-infer-keyword.ts` - infer 推断
- `03-mapped-types.ts` - 映射类型
- `04-template-literals.ts` - 模板字面量类型
- `05-type-guards.ts` - 类型守卫
- `06-type-challenges.ts` - 类型体操练习

## 学习目标

- 掌握条件类型的使用
- 理解 infer 关键字的作用
- 能够创建复杂的映射类型
- 解决常见的类型体操问题
