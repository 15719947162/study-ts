# 09 - 实战练习 (Practical Exercises)

通过实战项目综合运用 TypeScript 的各种特性。

## 知识点概览

1. **类型体操**: 复杂类型操作
2. **设计模式**: TypeScript 实现常见设计模式
3. **实战项目**: 完整的 TypeScript 项目示例

## 文件说明

- `01-type-challenges.ts` - 类型体操练习
- `02-design-patterns.ts` - 设计模式实现
- `03-mini-projects.ts` - 迷你项目实战

## 学习目标

- 掌握高级类型操作技巧
- 学会使用 TypeScript 实现设计模式
- 能够独立完成 TypeScript 项目开发
