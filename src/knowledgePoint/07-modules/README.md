# 07 - 模块与命名空间 (Modules & Namespaces)

TypeScript 支持 ES 模块和命名空间两种代码组织方式。

## 知识点概览

1. **ES 模块**: import/export 语法
2. **默认导出与命名导出**: export default vs export
3. **模块解析**: 相对路径与非相对路径
4. **命名空间**: namespace 关键字
5. **声明合并**: 模块与命名空间的合并
6. **类型导入**: import type 语法

## 文件说明

- `01-es-modules.ts` - ES 模块基础
- `02-module-resolution.ts` - 模块解析
- `03-namespaces.ts` - 命名空间
- `04-declaration-files.ts` - 声明文件

## 学习目标

- 掌握 ES 模块的导入导出
- 理解模块解析策略
- 学会使用命名空间组织代码
- 能够编写声明文件
